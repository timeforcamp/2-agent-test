# NEON ASTEROIDS

A single-file, browser-playable retro arcade game with neon vector graphics,
particle explosions, and Web Audio sound effects. Inspired by the 1979
Atari classic, with a synthwave twist.

## The Plan

Ship one self-contained `index.html` you can open directly in any modern
browser — no build step, no dependencies. Pure Canvas + Web Audio.

### Features
- Vector-style ship with thrust, rotation, and hyperspace
- Asteroids that split into smaller chunks when shot
- Bullet physics with screen-wrap
- Particle explosions
- Procedural Web Audio SFX (thrust, fire, boom)
- Wave-based progression + score + lives
- Pause + restart
- CRT-style scanline filter for the full retro vibe

### Crew
- **jam-man** (leader): plan, polish, final QA
- **neo** (worker): build the game in `index.html`

## How to play
Open `index.html` in a browser.
- **← / →** rotate
- **↑** thrust
- **Space** fire
- **Shift** hyperspace
- **P** pause
- **R** restart
